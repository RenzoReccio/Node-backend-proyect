RestaurantBack v2
=================

This is an API for a proyect using Nodejs, Express and MongoDB. I added CRUD opperation with MVC arquitechure


### Prerequisites

You need to install NodeJs in order to work:
https://nodejs.org/es/

### Installing

You need to install all npm dependencies

```
npm init
```

## Running the tests

To run the proyect
```
node server.js
```
